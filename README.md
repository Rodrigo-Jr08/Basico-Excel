# 📊 Pokémon Index - Excel Data Analysis Project

## 📖 Sobre o Projeto

Este projeto consiste no desenvolvimento de uma base de dados temática sobre Pokémon utilizando recursos avançados do Microsoft Excel. O objetivo foi aplicar técnicas de organização, tratamento, consulta e análise de dados, explorando funcionalidades frequentemente utilizadas em ambientes acadêmicos e profissionais.

A planilha foi estruturada para armazenar informações sobre Pokémon, seus movimentos, evoluções e relações de efetividade entre tipos, permitindo consultas rápidas e análises dinâmicas.

---

## 🎯 Objetivos

* Organizar e estruturar uma base de dados utilizando tabelas do Excel.
* Aplicar técnicas de tratamento e limpeza de dados.
* Utilizar funções matemáticas, estatísticas, lógicas e de texto.
* Implementar mecanismos de consulta e pesquisa de informações.
* Melhorar a visualização dos dados através de formatação e validações.
* Desenvolver uma Pokédex interativa para consulta de informações dos Pokémon.

---

## 📂 Estrutura da Planilha

### 🟢 Pokemon

Contém informações gerais dos Pokémon, incluindo:

* Número da Pokédex
* Nome
* Tipo(s)
* Estatísticas básicas
* Classificação por poder
* Região de origem
* Código identificador

### 🔵 Moves

Base de dados dos movimentos dos Pokémon contendo:

* Nome do movimento
* Tipo
* Poder
* Classificação do golpe
* Estatísticas de frequência

### 🟣 Evolution

Informações sobre as evoluções dos Pokémon:

* Pokémon inicial
* Evolução
* Descrições automáticas geradas por funções de texto

### 🔴 TypeChart

Tabela de efetividade entre os tipos de Pokémon:

* Very Effective
* Normal
* Not Very Effective
* Descrições geradas automaticamente

### 🟡 Pokedex Interativa

Sistema de consulta utilizando PROCX para localizar informações específicas de qualquer Pokémon presente na base.
