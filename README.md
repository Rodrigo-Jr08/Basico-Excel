# B-sico-Excel-e-formata-es
Esta planilha foi desenvolvida com o objetivo de organizar, analisar e consultar informações relacionadas ao universo Pokémon, utilizando recursos e funcionalidades do Microsoft Excel. A base de dados contém informações sobre Pokémon, movimentos (Moves), evoluções (Evolution) e relações de efetividade entre tipos (TypeChart).
